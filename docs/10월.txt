1. JSP 페이지 계산, 이름 설정

1) 초기 페이지(index)

1-1. 로그인안했을때 - 로그인, 회원가입 페이지로 이동 링크

1-2. 로그인했을때 - 로그아웃, 마이페이지(myPage.do) 이동 링크


2)로그인 페이지(loginForm)

-기능 미구현, 입력하고 submit 버튼만 있음


3) 회원가입 페이지(join)

3-1. 제대로 입력되면 완료 페이지로 이동(joinSuccess)

3-2. 값이 비어있으면 비었다고 알람 띄우기(알람 되는지는 확인 못해봄)



4) 회원가입 완료 페이지(joinSuccess)

4-1. 완료 메시지 출력

4-2. 처음 화면으로 돌아가기 링크



5) 마이페이지(myPage)

5-1. 개인정보 탭(탭 내용은 미구현)

5-2. 학생정보 탭

5-3. 공지사항 탭

5-4. 학사일정 탭

5-5. 수강신청 탭(수강신청 페이지로 이동, suGang)



6) 수강신청 페이지(suGang)

6-1. 강의목록 전체보기 탭

6-2. 장바구니(넣기랑 삭제) 탭

6-3. 게시판(게시판 페이지로 이동, board)


7) 게시판(board)

7-1. 수강후기(미구현)

7-2. 게시판 전체 보기(미구현)


8) 관리자 페이지(admin)

*로그인 페이지에서 관리자 아이디, 비밀번호 일때만 접근

8-1. 학생조회 - 해당 링크로 이동

8-2. 학생정보수정 - 해당 링크로 이동

8-3. 강의 추가, 삭제 - 해당 링크로 이동


9) 학생조회 페이지(studentSearch)

미구현(전체조회와 상세조회, 게시글 목록 조회랑 게시글 조회 기능 참고해서 만들기)


10) 학생정보수정 페이지(studentInfo)

미구현

수정시 팝업창 띄우기


11) 강의수정 페이지(lecture)

11-1) 수정

11-2) 삭제

수정시 팝업창 띄우기


필요한 페이지 약 11개?
